Definitions to Key words in JavaScript

JavaScript

an object-oriented computer programming language commonly used to create interactive effects within web browsers.

operator

variable

Variables can be thought of as named containers. You can place data into these containers and then refer to the data simply by naming the container. Before you use a variable in a JavaScript program, you must declare it. Variables are declared with the var keyword as follows.

Variable naming conventions

Coding conventions are style guidelines for programming

Functions

Functions are one of the fundamental building blocks in JavaScript. A function is a JavaScript procedure—a set of statements that performs a task or calculates a value. To use a function, you must define it somewhere in the scope from which you wish to call it.

Strings

a string is traditionally a sequence of characters, either as a literal constant or as some kind of variable. The latter may allow its elements to be mutated and the length changed, or it may be fixed (after creation).

Methods

A JavaScript method is a property containing a function definition.

Arguments

The arguments object is a local variable available within all functions. You can refer to a function's arguments within the function by using the arguments object.

Parameters

a numerical or other measurable factor forming one of a set that defines a system or sets the conditions of its operation.

Chaining methods

Method chaining is a technique that can be used to simplify code in scenarios that involve calling multiple functions on the same object consecutively.

Boolean

a binary variable, having two possible values called “true” and “false.”

Undefined

In JavaScript there is Undefined (type), undefined (value) and undefined (variable). Undefined (type) is a built-in JavaScript type. undefined (value) is a primitive and is the sole value of the Undefined type. Any property that has not been assigned a value, assumes the undefined value.

Nan

The NaN property represents "Not-a-Number" value. This property indicates that a value is not a legal number

Escape

escape() function computes a new string in which certain characters have been replaced by a hexadecimal escape sequence.

Alert

The alert() method displays an alert box with a specified message and an OK button.An alert box is often used if you want to make sure information comes through to the user.

Comments

JavaScript comments can be used to explain JavaScript code

jQuery

Query is a cross-platform JavaScript library designed to simplify the client-side scripting of HTML. It is free, open-source software using the permissive MIT License.

Attributes

Attributes provide additional information about HTML elements.

Return

In JavaScript, return statements cease execution in a function and return a value to the caller; JavaScript functions require an explicit return statement for returning the result of an expression (the value) from a function.
